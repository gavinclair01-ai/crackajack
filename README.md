# Cracka Jack — Landing Page

Pre-order landing page for the Cracka Jack caravan front jack.

Live at: https://gavinclair.github.io/crackajack/ (once deployed)

## Deploy to GitHub Pages

1. Create repo on GitHub: `crackajack` (public)
2. Push this folder
3. Go to Settings → Pages → Deploy from branch → main → / (root)
4. Done — live in ~60 seconds
